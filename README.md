# Buddyscript

A modern, full-stack social media web application built with Next.js, featuring secure authentication, a dynamic post feed with text and image support, and interactive capabilities like nested comments and post likes.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Technologies Used

- Next.js
- Tailwind CSS
- MongoDB
- Cloudinary (for image uploads)
